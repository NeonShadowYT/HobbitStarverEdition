Eng
----------------------------------------------------------
This is a very useful template to edit the .Export files for the Hobbit 2003 game.

In this folder you just put all you .export files and modify them here. You can simply drag'n'drop a .export file on decompress export.bat and it will become .export.txt.

Then you just edit all the files needed and run export_move_to_game.py. Make sure to change the path inside the export_move_to_game.py to the one which 
corresponds to unpacked level folder. (for example C:\Games\Sierra\The Hobbit(TM)\Levels\CH00_DREAMWORLD)

Use python to run this script ( while installing put tick on ADD TO PATH ): python.org

The best way of utiliztying this is using Unpacked Edition of the Hobbit: https://github.com/hobbit-kingdom/hobbit-versions/releases

IMPORTANT: if you also are using Level Editor make sure to save everything in the editor and then put a new .export and 
DECOMPRESSING it before you run export_move_to_game.py again!!!

In the Sub folder you can modify subtitle files like .bin.

My recomendation is to rename this template to smth like L7 (BooB) or L1 (Unexpected party). To easily find what this folder corresponds to.

Rus
----------------------------------------------------------
Это очень полезный шаблон для редактирования файлов .Export для игры Hobbit 2003.

В эту папку вы просто помещаете все свои файлы .export и изменяете их здесь. Вы можете просто перетащить .export файл на распаковку export.bat, и он превратится в .export.txt.

Затем вы просто редактируете все необходимые файлы и запускаете export_move_to_game.py. Не забудьте изменить путь внутри export_move_to_game.py на тот, который 
соответствует папке с распакованным уровнем. (например, C:\Games\Sierra\The Hobbit(TM)\Levels\CH00_DREAMWORLD).

Используйте python для запуска этого скрипта (при установке поставьте галочку в ADD TO PATH): python.org

Лучший способ использовать это - использовать распакованное издание Хоббита: https://github.com/hobbit-kingdom/hobbit-versions/releases

ВАЖНО: если вы также используете Level Editor, обязательно сохраните все в редакторе, а затем создайте новый .export и 
ДЕКОМПРЕССИРУЙТЕ его, прежде чем снова запускать export_move_to_game.py!!!

В папке Sub вы можете изменять файлы субтитров, например .bin.

Я рекомендую переименовать этот шаблон во что-нибудь вроде L7 (BooB) или L1 (Unexpected party). Чтобы легко найти, чему соответствует эта папка.
